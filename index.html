Atividade Prática

Passo a passo de como realizar esta mesma atividade.

1º Passo

Comece criando um site básico no seu editor de código. O site deve ter pelo menos uma página (index.html) e utilizar HTML, CSS e Bootstrap. Você é livre para definir o conteúdo e estilo do site.

2º Passo

Após criar o site, crie um novo repositório no GitHub. Nomeie o repositório de acordo com o propósito do seu site (por exemplo, “meu-portfolio”).

3º Passo

No VSCode, adicione o repositório que você acabou de criar como o repositório remoto do seu projeto usando o comando git remote add origin [URL do seu repositório].

4º Passo

Faça o push do seu projeto para o GitHub usando o comando git push origin main.

5º Passo

Verifique se o seu projeto aparece no repositório no GitHub.

6º Passo

Ative o GitHub Pages para o seu repositório. Vá para Settings > Pages > Source, escolha a branch “main” e clique em “Save”.

7º Passo

Aguarde alguns minutos para o GitHub publicar o seu site. Após a publicação, copie o URL fornecido pelo GitHub e verifique se o site está funcionando corretamente.

8º Passo

Compartilhe o URL do seu site com seus colegas de classe.
